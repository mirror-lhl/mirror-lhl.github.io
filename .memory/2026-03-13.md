## Payment Page CSP Fix
- 根本原因：内联 onclick 被 CSP script-src 'none' 策略阻止，同时存在 initiatePayment 函数名对不上的问题
- 修复方案：移除所有 HTML 内联 onclick 属性，在 DOMContentLoaded 里统一用 addEventListener 绑定
- 新增 initiatePayment() 别名函数保持向后兼容
- 涉及按钮：payBtn、method-alipay、method-wechat、tab-alipay、tab-wechat、copyOrderBtn、copyRemarkBtn、closeQrBtn、cancelQrBtn、confirmBtn、closeSuccessBtn
