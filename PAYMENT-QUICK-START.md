# 🚀 个人收款设置 - 快速开始指南

## ⏱️ 预计完成时间: 5分钟

---

## 📋 第1步: 准备收款二维码 (2分钟)

### 支付宝收款二维码
1. 打开支付宝APP
2. 点击右上角 "+" 号
3. 选择"收钱"
4. 点击"保存收款码"
5. 将保存的图片重命名为 `alipay-qr-code.jpg`
6. 将图片放入项目的 `images/` 文件夹

### 微信收款二维码
1. 打开微信APP
2. 点击"我" → "服务" → "收付款"
3. 点击"二维码收款"
4. 长按保存图片到相册
5. 将保存的图片重命名为 `wechat-qr-code.jpg`
6. 将图片放入项目的 `images/` 文件夹

---

## ✏️ 第2步: 填写配置信息 (2分钟)

打开文件: `js/payment-config.js`

将以下信息替换为您的真实信息:

```javascript
const PAYMENT_CONFIG = {
    alipay: {
        qrCodePath: 'images/alipay-qr-code.jpg',
        account: '您的支付宝账号',           // ← 替换这里
        name: '您的真实姓名',                // ← 替换这里
        note: '请扫描二维码支付,支付时请备注订单号以便确认'
    },
    
    wechat: {
        qrCodePath: 'images/wechat-qr-code.jpg',
        account: '您的微信号',               // ← 替换这里
        name: '您的昵称',                   // ← 替换这里
        note: '请扫描二维码支付,支付时请备注订单号以便确认'
    }
};
```

### 配置示例:

```javascript
const PAYMENT_CONFIG = {
    alipay: {
        qrCodePath: 'images/alipay-qr-code.jpg',
        account: 'zhangsan@example.com',     // 您的支付宝账号
        name: '张三',                        // 您的真实姓名
        note: '请扫描二维码支付,支付时请备注订单号以便确认'
    },
    
    wechat: {
        qrCodePath: 'images/wechat-qr-code.jpg',
        account: 'zhangsan_wx',              // 您的微信号
        name: '张三的小店',                  // 您的昵称
        note: '请扫描二维码支付,支付时请备注订单号以便确认'
    }
};
```

## 🧪 第3步: 测试支付页面 (1分钟)

### 测试步骤:
1. 在浏览器中打开: `payment-personal.html`
2. 点击"生成支付二维码"按钮
3. 选择支付宝或微信支付
4. 检查二维码是否正确显示
5. 检查收款信息是否正确

### 如果二维码不显示:
- 检查 `images/` 文件夹中是否有二维码图片
- 检查图片文件名是否正确
- 检查 `js/payment-config.js` 中的路径是否正确

## 💡 使用说明

### 用户支付流程:
1. 用户在产品页面点击"订阅/购买"
2. 跳转到支付页面
3. 选择支付方式 (支付宝/微信)
4. 点击"生成支付二维码"
5. 扫描二维码完成支付
6. **重要**: 支付时备注订单号
7. 点击"已完成支付"按钮
8. 系统提示提交成功

### 商家确认流程:
1. 检查支付宝/微信收款记录
2. 核对订单号是否匹配
3. 确认金额是否正确
4. 手动更新订单状态
5. 为用户开通服务权限

## 🔔 重要提醒

### 支付备注:
- ⚠️ **务必提醒用户备注订单号**
- 用户支付时忘记备注,无法确认订单
- 建议在支付页面多次强调

### 订单确认:
- 需要人工检查收款记录
- 建议每天固定时间确认订单
- 保存好每笔交易的截图
- 及时为用户开通服务

### 安全建议:
- 不要接收大额转账 (>¥500)
- 如发现可疑交易,立即停止
- 定期检查收款记录
- 保护个人隐私信息

## ❓ 常见问题

### Q: 二维码显示不出来怎么办?
A: 检查以下几点:
- 图片是否放在 `images/` 文件夹中
- 图片文件名是否正确
- 路径是否正确 (`images/alipay-qr-code.jpg`)

### Q: 用户忘记备注订单号怎么办?
A: 
- 检查收款时间
- 对比金额和时间
- 联系用户确认
- 更新订单记录

### Q: 可以收款大额金额吗?
A: 
- 个人收款建议单笔不超过¥500
- 如需大额收款,建议申请商家账号
- 或使用银行转账方式

### Q: 如何升级到正式支付系统?
A: 
- 申请支付宝商家版: https://b.alipay.com/
- 申请微信支付商户: https://pay.weixin.qq.com/
- 对接官方支付接口
- 需要后端开发支持
