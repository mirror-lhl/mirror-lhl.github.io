# 收款二维码图片文件夹

此文件夹用于存放您的收款二维码图片。

## 需要添加的文件:

1. **支付宝收款码**
   - 文件名: `alipay-qr-code.jpg`
   - 如何获取: 打开支付宝APP → 收钱 → 保存收款码
   - 将图片重命名为 alipay-qr-code.jpg 并放入此文件夹

2. **微信收款码**
   - 文件名: `wechat-qr-code.jpg`
   - 如何获取: 打开微信APP → 我 → 服务 → 收付款 → 二维码收款 → 长按保存
   - 将图片重命名为 wechat-qr-code.jpg 并放入此文件夹

## 注意事项:

- 图片格式: JPG 或 PNG
- 图片大小: 建议 500x500 像素
- 文件名必须完全匹配(区分大小写)
- 确保图片清晰可扫描
